#C_project

소스코드:
  - main.c : main함수 있는 c파일
  - dir.h : 딕셔너리 관련 헤더파일
  - bitmap.h : 주요 실행 코드 관련 헤더파일 
------ 
딕셔너리 구조:
  - image_file 딕셔너리
    - 이미지파일1
    - 이미지파일2
    - 이미지파일3
    - ...
    
  - text_file 딕셔너리
    - 이미지가 텍스트로 변환된 파일
  - exe실행 프로그램
------
설명 : 
  1.  image_file 딕셔너리에 있는 이미지 파일을 고른뒤 
  2.  프로그램이 text_file 딕셔너리에 변환된 텍스트 파일 생성
